Our Purpose is Two-fold: 1) to facilitate language learning in a casual environment and 2) promote responsible travel so that one may have more meaningful experiences abroad.

Our main activity consists of Bi-Weekly Language Exchange Meetings that take place in a different cafe every quarter. How it Works: Participants start by checking-in. Each person then receives a name tag that displays the languages they speak or would like to learn. We begin the meetings in English so that everyone can understand each other. As participants join the event, we open up different language groups. The languages we have represented are Spanish, Mandarin, Danish, Lithuanian, Japanese, and Arabic (with more to come). All language levels are welcome, especially beginners! This is also a great opportunity for non-native speakers to improve their English.
